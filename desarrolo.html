Conceptos de Software e Ingeniería del Software

1. ¿Qué es el Software?

El software se define como el conjunto de instrucciones, programas, procedimientos y datos que permiten que un dispositivo electrónico, como una computadora, un teléfono o un sistema embebido, realice tareas específicas. El software no es tangible: no se puede tocar, pero es indispensable para que el hardware funcione.

El software actúa como intermediario entre el usuario y el hardware. Gracias a él, el usuario puede comunicarse con la máquina mediante interfaces gráficas, programas de propósito específico o sistemas complejos.

Categorías del Software

  • Software de sistema: Controla y gestiona el hardware. Ejemplos: Windows, Linux, Android.
  • Software de aplicación: Programas para realizar tareas específicas como editar documentos, navegar por internet o jugar.
  • Software de programación: Herramientas para crear software, como editores, compiladores y entornos de desarrollo.

2. ¿Cómo Funciona el Software?

El software funciona como un conjunto de instrucciones secuenciales que el procesador interpreta y ejecuta. Este proceso implica una serie de pasos:

  1. El usuario realiza una acción (clic, comando, interacción).
  2. El software traduce la acción en instrucciones.
  3. La CPU recibe instrucciones codificadas.
  4. La máquina ejecuta la operación.
  5. El resultado vuelve al usuario en forma visual o auditiva.

Este ciclo ocurre millones de veces por segundo, permitiendo el funcionamiento fluido de programas complejos.

3. Máquinas Virtuales

Una máquina virtual (VM) es un entorno que simula un sistema informático completo. Permite ejecutar sistemas operativos completos o procesos aislados sin afectar la máquina física.

Máquinas Virtuales de Sistema

Simulan hardware completo. Se usan para pruebas, compatibilidad, servidores o aislar sistemas.

  • VMware
  • VirtualBox
  • Hyper-V

Máquinas Virtuales de Proceso

Ejecutan procesos o aplicaciones específicos de forma independiente. Ejemplo fundamental: la Java Virtual Machine (JVM).

4. Proceso para Ejecutar un Programa en Java

Java utiliza un proceso en dos fases:

  1. El programador escribe código fuente en .java.
  2. El compilador javac lo transforma en bytecode (.class).
  3. La JVM interpreta el bytecode y ejecuta el programa en cualquier máquina.

Este mecanismo permite la famosa frase: “Write once, run anywhere”.

5. Clasificación de Lenguajes

Los lenguajes se clasifican según su nivel de abstracción del hardware.

Lenguajes de Bajo Nivel

Ejemplos: lenguaje máquina, ensamblador. Son rápidos y eficientes, pero difíciles de programar.

Lenguajes de Alto Nivel

Más fáciles de comprender. Ejemplos:

  • Python
  • Java
  • C#
  • JavaScript

7. Conceptos de Código

Código Fuente

Texto escrito por el programador.

Código Objeto

Versión intermedia generada por el compilador.

Código Ejecutable

Archivo final que el sistema operativo puede ejecutar.
